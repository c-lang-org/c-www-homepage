# Contact

- Contact information to **Office Holders** can be found on the [contacts](https://www.open-std.org/JTC1/SC22/WG14/www/contacts.html) page on the committee's website.

- If you want to make a **proposal** to the language see [contributing](https://www.open-std.org/jtc1/sc22/wg14/www/contributing) page on the committee's website.

- If you want to contact an individual member seek the contact informations on their submitted documents.

- If you want to share a suggestion for this website, send us an e-mail to [webmaster@c-language.org](mailto:webmaster@c-language.org).
  
  <x-cmt>WARNING: In majority of cases, messages with self promotion will be ignored. Please, engage with the [community](/community) first.</x-cmt>

- There is also an [official GitHub profile](https://github.com/c-lang-org).
