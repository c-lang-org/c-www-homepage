# Community

The C programming community spans multiple platforms and formats. Here are some of the most active communities where C programmers gather to share knowledge and collaborate.

## Organizations

**[ACCU](https://accu.org/)**

While nowadays its interests encompass a wide range of languages and topics, previously it was known as *Association of C and C++ Users*.

**[C Board](https://cboard.cprogramming.com/c-programming/?daysprune=-1)**

Forum accompanying [Cprogramming.com](https://cprogramming.com) website.

**[C/C++ Users Group](http://www.hal9k.com/cug/)**

A (now historical) group focused on low-cost distribution of open-source C/C++ code. CUG also launched *C/C++ Users Journal*. Some of their CD-ROMs can be found archived on [Internet Archive](https://archive.org/search?query=%22C+Users+Group%22).

**[Codidact](https://software.codidact.com/categories/38/tags/3296)**

Software development Q&A platform with dedicated C tag.

**[Comprehensive C Archive Network](https://ccodearchive.net/)**

Beside the repository itself, there are [mailing list](https://lists.ozlabs.org/listinfo/ccan) and #ccan channel on Freenode. <x-cmt>CCAN is inspired by Perl's CPAN</x-cmt>

**[DEV Community](https://dev.to/t/c)**

Social network for software developers with dedicated C tag.

## Other Platforms

**[Lobsters](https://lobste.rs/t/c)**

A link aggregation and discussion board with tag dedicated to C programming. <x-cmt>posting only after an invite-only registration</x-cmt>

**[Meetup](https://www.meetup.com/topics/c-programming-language/)**

This social networking platform for creating local groups and events has a dedicated *C & C++* topic tag.

**Newsgroups**

There are prominent newsgroups about C language and adjacent topics:

- [comp.lang.c](news:comp.lang.c) <x-cmt>[clc-wiki](https://clc-wiki.net/wiki/Main_Page), [novaBBS](https://www.novabbs.com/devel/thread.php?group=comp.lang.c), [Google Groups](https://groups.google.com/g/comp.lang.c)</x-cmt>
- [comp.std.c](news:comp.std.c) <x-cmt>[novaBBS](https://www.novabbs.com/devel/thread.php?group=comp.std.c), [Google Groups](https://groups.google.com/g/comp.std.c)</x-cmt>

**[Reddit](https://www.reddit.com/r/C_Programming/)**

Subreddit dedicated to discussion of the C programming language with over 250k members.

**[Stack Overflow](https://stackoverflow.com/questions/tagged/c)**

The largest technical Q&A site has over 400,000 questions tagged with C.
